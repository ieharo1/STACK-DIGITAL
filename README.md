Portafolio completo de tecnologías digitales que abarca desarrollo web, backend, bases de datos, cloud computing y ciencia de datos. Desarrollado por **Isaac Esteban Haro Torres**.

---

## 📝 Descripción

Proyecto educativo y de portafolio que demuestra competencias en múltiples tecnologías modernas del desarrollo de software.

### ¿Qué hace este proyecto?

- **Full Stack Development**: Demuestra habilidades en frontend y backend
- **Cloud & DevOps**: Configuración y despliegue en la nube
- **Data Science**: Análisis de datos y visualización
- **Machine Learning**: Modelos predictivos básicos
- **APIs REST**: Diseño e implementación de servicios web

## 🔧 Cómo Usar Este Portafolio

### Explorar proyectos

```bash
# Clonar y explorar
git clone https://github.com/ieharo1/STACK-DIGITAL.git
cd STACK-DIGITAL

# Ver estructura
tree -L 2
```

### Ejecutar ejemplos

```bash
# Backend Node.js
cd backend/node-api
npm install
npm run dev

# Python API
cd backend/python-api
pip install -r requirements.txt
uvicorn main:app

# Docker
cd docker
docker-compose up
```
